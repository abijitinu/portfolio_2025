The Context

SITA, the world's leading air transport IT and communications specialist, had operations scattered across multiple disjointed portals. This fragmentation led to inconsistent user experiences and inefficient workflows for both internal teams and external partners.

The Brief

The goal was to centralize all operations into a single, unified Web Portal. To achieve this, we needed to build a robust, scalable Design System that would serve as the foundation for all current and future applications.

Key Requirements

  • Reusability: Maintain a comprehensive design system allowing teams to reuse components to build necessary screens efficiently.
  • Flexibility: Components must be flexible enough to accommodate undefined future scenarios and use cases.
  • Scalability: Screens and components need to be scalable, capable of handling increased information density without breaking the layout.

The Audience

The primary users of the Design System are SITA's Internal Product Teams (Developers, Product Managers, Designers) and External Partners who build and integrate applications for the air transport industry.

The Process

We started by auditing existing portals to identify common patterns and inconsistencies. We then adopted an Atomic Design methodology, creating a library of reusable components ranging from basic atoms (buttons, inputs) to complex organisms (tables, dashboards).

Regular workshops with development teams ensured that the components were technically feasible and easy to implement.

The Solution

The SITA Design System provides a comprehensive set of guidelines, components, and patterns. It ensures:

  • Unified Experience: A consistent look and feel across all SITA applications.
  • Efficiency: Pre-built components allow developers to focus on logic rather than UI styling.
  • Accessibility: Built-in accessibility standards ensure the portal is usable by everyone.

The Outcome

The Design System significantly accelerated development cycles, reducing the time to build new screens by 50%. It also ensured a consistent brand experience across all SITA applications, improving user trust and adoption.
